## Star map for the northern celestial hemisphere, magnitude 1–6
- approximately ~4,000 stars
- azimuthal equidistant, North Celestial Pole centered projection
- Symbols:
  * Mag ≤ 2: 4 px square
  * Mag 3–4: 3 px square
  * Mag 5: 2 px square
  * Mag 6: single-pixel dot

You can use https://splitter.imageonline.co/ to split images if you do not have [imagemagick](https://imagemagick.org) installed.
